# 우리동네스마트건강지도 기획안

## 프로젝트명
우리동네스마트건강지도

## 문제정의
지역주민들은 폭염, 한파, 미세먼지, 감염병 유행 같은 기후 및 보건 위기 상황에서 특정 시기에 병원 이용이 급격히 늘어나며 진료 대기시간이 길어지는 문제를 겪고 있다. 특히 아이를 둔 보호자, 만성질환자, 고령층은 적절한 병원을 빠르게 찾기 어려워 불필요한 이동과 장시간 대기를 반복하게 된다. 이 문제를 해결하기 위해 지역 단위의 병원 혼잡도, 예상 대기시간, 감염 위험 정보를 통합해 안내하는 서비스가 필요하다.

## 핵심기능
1. 실시간 병원 혼잡도 지도
지역 내 병원별 현재 대기 인원, 혼잡 상태(여유/보통/혼잡)를 지도에서 한눈에 확인한다.

2. 진료 대기시간 예측
시간대, 요일, 기후 데이터, 감염병 동향을 반영해 병원별 예상 대기시간을 제공한다.

3. 증상/상황 기반 병원 추천
사용자가 증상, 연령, 이동 가능 거리, 진료 과목을 입력하면 가장 빠르게 진료 가능한 병원을 추천한다.

4. 감염 위험 알림
지역 감염병 경보, 기후 위험(폭염/한파/미세먼지) 알림을 제공하고 예방 행동 가이드를 안내한다.

5. 원격 대기 등록 및 방문 타이밍 안내
대기 가능 병원에 대해 사전 대기 등록을 지원하고, 방문 권장 시간을 알려준다.

## 기대효과
1. 주민 관점
평균 진료 대기시간과 불필요한 이동 시간이 줄어들고, 위기 상황에서도 더 빠르게 적정 진료를 받을 수 있다.

2. 병원 관점
내원 환자가 특정 시간과 특정 병원에 집중되는 현상이 완화되어 운영 효율이 개선된다.

3. 지역 보건 관점
감염병 및 기후 취약 지역을 조기에 파악하여 보건소와 지자체의 대응 속도를 높일 수 있다.

## 핵심화면
1. 메인 지도 화면
- 내 주변 병원 보기 버튼
- 예상 대기시간 순 정렬 버튼
- 감염 위험 레이어 켜기/끄기 버튼
- 내 증상으로 병원 찾기 버튼
- 응급실/야간진료 바로가기 버튼

2. 병원 상세 화면
- 현재 대기 인원 보기 버튼
- 예상 대기시간 확인 버튼
- 원격 대기 등록 버튼
- 길찾기 버튼
- 전화 연결 버튼

3. 알림 설정 화면
- 감염병 알림 받기 버튼
- 기후 위험 알림 받기 버튼
- 자주 가는 병원 알림 설정 버튼

## 페르소나
1. 김지현(39세, 초등학생 자녀 2명 보호자)
아이가 갑자기 아플 때 가까운 병원을 찾지만 대기시간이 길어 어려움을 겪는다. 빠른 진료가 가능한 병원을 즉시 확인하고 싶어 한다.

2. 박영수(67세, 고혈압/당뇨 만성질환자)
정기 진료를 위해 병원을 자주 방문한다. 장시간 대기가 체력적으로 부담되어 혼잡하지 않은 시간대 정보를 필요로 한다.
